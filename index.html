# mt_attributes Contains all attribute of an entity (e.g. shader) Param | Description ------------------------|--------------------------------------------------- **cnt**` | Number of attributes in struct **types** | Type for each attribute (e.g. MT_FLOAT_ATTR) **values** | All values for all attr packed together **values_byte_offset** | Byte-offset of each attr in **values** **values_num_bytes** | Total size in bytes of **values** **names** | Name of each attribute ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ C typedef struct mt_attributes { size_t cnt; mt_attr_type* types; char* values; size_t* values_byte_offset; size_t values_num_bytes; char** names; } mt_attributes; ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
